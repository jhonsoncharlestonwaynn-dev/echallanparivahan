# Project Documentation

## Overview
This project is designed for the eChallan system, focusing on providing seamless functionalities for various user services. Key features include:

### Bulk SMS Sending
- Ability to send bulk SMS notifications to users regarding their challan status or other relevant updates.
- Easily customizable templates for SMS content.

### APK Download
- Provides a straightforward method for users to download the mobile application directly from the website.
- The APK is updated regularly to ensure users receive the latest features and security updates.

## Usage
1. Access the application through the provided web link.
2. For bulk SMS, navigate to the SMS section, input user numbers, and customize your message.
3. Download the APK by clicking on the download button on the homepage.

## Installation
Ensure that you have the required dependencies and setup to run the project effectively. For detailed instructions, refer to the setup guide in the documentation.
